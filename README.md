# 📄 AI Resume Reviewer

An AI powered Resume Reviewer built with Streamlit and Groq API.

## Features
- Paste your resume and get instant AI feedback
- Shows Strengths, Weaknesses and Improvements
- Gives overall rating out of 10
- Built with Llama 3.3 model via Groq

## Tech Used
- Python
- Streamlit
- Groq API
- Llama 3.3 model

## How to Run
pip install streamlit groq
streamlit run app.py
